# RGPV Result Analyzer (C)

A simple command-line based **RGPV Result Analyzer** built using the **C programming language**.

This project analyzes student exam results to calculate important statistics such as **average marks, highest marks (topper), and overall performance insights**.

The goal of this project is to practice **core programming concepts while solving a real-world problem like academic result analysis.**

---

## Features

• Add student marks  
• Display all student results  
• Calculate average marks  
• Identify the topper (highest marks)  
• Menu-driven command-line interface  

---

## Technologies Used

• C Programming Language  
• Arrays  
• Functions  
• Loops  
• Conditional Statements  
• Switch Case
