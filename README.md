# RGPV Result Analyzer (C & Python)

A simple command-line based **RGPV Result Analyzer** built using **C and Python**.

This project analyzes student exam results to calculate important statistics such as **average marks, highest marks (topper), and overall performance insights**.

The goal of this project is to demonstrate how the **same real-world problem can be solved using two different programming languages while strengthening programming fundamentals.**

---

## Features

- Add student marks
- Display all student results
- Calculate average marks
- Identify the topper (highest marks)
- Menu-driven command-line interface

## How to Run

### Run the C Version

Compile the program

```
gcc result_analyzer.c -o result
```

Run the program

Linux / Mac
```
./result
```

Windows
```
result.exe
```

---

### Run the Python Version

Run the program using Python

```
python result_analyzer.py
```

or

```
python3 result_analyzer.py
```
